# MotetAI

## 📋 概要

```
好きな人と付き合いたい！この恋をなんとしても成就させたい！！
...けど、モテるやつって何をもってしてモテるやつなんだろう？

恋人を「作りたい」、その思いを胸に遂に本作を手に取ったあなた。
友達や家族、あわよくば好きな子と一緒に本作を遊び、シナリオに合わせて臨機応変にモテる行動を取り、勝利を勝ち取ろう！
もしかしたら、このゲームを通して恋愛マスターになれちゃうかも...？
```
